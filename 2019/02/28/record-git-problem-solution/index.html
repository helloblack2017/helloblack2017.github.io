问题背景

执行hexo d时出现以下问题：

前置知识

CR 回车
LF 换行
Windows/Dos CRLF \r\n
Linux/Unix LF \n
MacOS CR \r

解决方法

打开命令行进行设置，在Windows下开发，建议设置autocrlf为true。
如果文件编码是UTF8并且包含中文文字，那还是把autocrlf设置为false，并且把所有文件转换为Linux编码（即LF\n），开启safecrlf检查。
