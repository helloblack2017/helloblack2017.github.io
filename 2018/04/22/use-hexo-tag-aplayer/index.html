选项
默认值
描述






id
必须值
歌曲 id / 播放列表 id / 相册 id / 搜索关键字




server
必须值
音乐平台: netease, tencent, kugou, xiami, baidu




type
必须值
列表播放模式, circulation, random, single, order




autoplay
true
自动播放，移动端浏览器暂时不支持此功能




mutex
true
该选项开启时，如果同页面有其他 aplayer 播放，该播放器会暂停




listmaxheight
340px
播放列表的最大长度




preload
auto
音乐文件预载入模式，可选项： none, metadata, auto




theme
#ad7a86

播放器风格色彩设置
